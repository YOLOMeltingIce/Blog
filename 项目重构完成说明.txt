=====================================================
二、如何使用新系统
=====================================================

步骤 1: 安装依赖
-----------------
npm install

步骤 2: 配置 Notion 数据库
-----------------
参考《Notion数据库字段说明.txt》文件，在 Notion 中配置三个数据库：
1. 论文读后感数据库
2. 项目经历数据库
3. AI 时间轴数据库

每个数据库的必需字段和推荐字段都已在说明文件中详细列出。

步骤 3: 配置环境变量
-----------------
在项目根目录创建 .env.local 文件（或直接修改 scripts/fetch-notion-data.js）：

NOTION_TOKEN=你的_Notion_Integration_Token
NOTION_PAPERS_DATABASE_ID=论文数据库ID
NOTION_PROJECTS_DATABASE_ID=项目数据库ID
NOTION_AI_TIMELINE_DATABASE_ID=AI时间轴数据库ID

步骤 4: 获取数据
-----------------
npm run fetch-data

这将从 Notion 获取数据并保存到 data/ 目录。

步骤 5: 运行开发服务器
-----------------
npm run dev

访问 http://localhost:3000 查看效果。

步骤 6: 构建生产版本
-----------------
npm run build
npm start

=====================================================
六、常见问题 FAQ
=====================================================

Q1: 样式没有生效怎么办？
A: 
1. 确保已安装新依赖：npm install
2. 清除 .next 缓存：删除 .next 文件夹
3. 重启开发服务器：npm run dev

Q2: Notion 数据获取失败？
A:
1. 检查 NOTION_TOKEN 是否正确
2. 检查数据库 ID 是否正确
3. 确保 Integration 有数据库访问权限
4. 查看控制台错误信息

Q3: 字段名不匹配怎么办？
A:
脚本支持中英文字段名自动匹配，参考《Notion数据库字段说明.txt》
中的"字段名称别名"部分。

Q4: 如何添加新的页面？
A:
1. 在 src/app/ 下创建新文件夹
2. 添加 page.tsx 文件
3. 在 Navigation.tsx 中添加导航项

Q5: 移动端样式有问题？
A:
检查响应式类名：
- md: 中等屏幕及以上
- sm: 小屏幕及以上
- 默认：移动端
